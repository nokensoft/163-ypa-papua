# YPA Papua Website
Website profil YPA Papua dengan arsitektur multi-halaman statis, navigasi konsisten, mode tema terang/gelap, serta kontrol translate di topbar.

## Live Demo
https://nokensoft.github.io/163-ypa-papua/

## Fitur Utama Website
- Multi-page static layout dengan header dan footer konsisten di semua halaman.
- Light/Dark mode toggle pada topbar.
- Custom translate control (English/Indonesia) terintegrasi Google Translate.
- Mega menu dropdown untuk menu **About**.
- Navigasi responsif (desktop + mobile menu).
- Page loader menggunakan logo utama.
- Floating action button WhatsApp + back-to-top.
- CTA strip **TITIP Program** sebelum footer di semua halaman.
- Konten home terbaru:
  - Postingan Terbaru
  - Section **A quick look at YPA Papua** dengan layout 2 kolom seimbang (teks kiri + gambar kanan)
  - Gallery highlights format Stories/Reels/Live
  - Shop Preview 4 kolom (kartu produk persegi)

## Daftar Navigasi & Halaman
- **Home** (`index.html`)  
  Halaman utama berisi hero, postingan terbaru, section sekilas YPA Papua 2 kolom, galeri highlights, shop preview, dan CTA TITIP Program.
- **Profile** (`profile.html`)  
  Menjelaskan who we are dan what we do.
- **About** (`about.html`)  
  Latar belakang yayasan, konteks organisasi, dan arah pelayanan.
- **Mission** (`mission.html`)  
  Pilar misi, fokus program, dan prioritas dampak.
- **TITIP** (`titip.html`)  
  Informasi program TITIP serta pengembangan peserta.
- **Blog** (`blog.html`)  
  Kumpulan artikel dan pengumuman.
- **Gallery** (`gallery.html`)  
  Dokumentasi visual kegiatan dan event.
- **Donation** (`donation.html`)  
  Informasi dukungan/donasi untuk program pelayanan.
- **Shop** (`shop.html`)  
  Katalog produk (Let Art, merchandise, dll).
- **Contact** (`contact.html`)  
  Informasi kontak dan form komunikasi.
